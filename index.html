Qu'est-ce que le MARKDOWN ?

Le Markdown est un langage de balisage léger. Il offre une syntaxe facile à lire et à écrire. Il a été créé en 2004 par John Gruber et Aaron Swartz.

Mise en oeuvre
Le Markdown s'utilise avec différents langages de programmation : Perl, PHP, Ruby, Java, Python, C#, etc. Il se termine par l'extension *.md ou .markdown.

Utilisation
Le Markdown est très polyvalent, il permet de créer des listes, des tableaux, des liens, insérer des images, etc. Ci-dessous, différents exemple d'application du Markdown.

Afficher du code :

Pour afficher du code en format brut (non interpreté) :

  • faire tab a chaque ligne de code

  • quatre espaces avant chaque ligne de code

  • utiliser trois bacticks avant votre bloc de code ( ` ) :

    ``` votre code ici```

Vous pouvez même afficher les couleurs en ajoutant le nom du langage après les backticks : 

```js "votre code" ```
